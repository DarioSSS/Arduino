menu_I2C

To projekt na Arduino Mega, do którego podłączone są po I2C:
- Keypad przez expander PCF8574,
- zegar RTC 3231
- ekran LCD 20x4 z expanderem PCF8574
a wszystko to sterowane przez menu textowe.
